# 🤖 6-DOF Robot Arm Educational Kit

> 🚀 An Interactive Robotics Learning Platform using **ESP32, ESP32-CAM, Python, IoT Control, and Digital Twin** integrated with an interactive GUI

---

## 📌 Overview

This project presents a **complete educational kit** for a **6-DOF robotic arm**, designed to help students understand robotics concepts in a practical and interactive way.

### 🔗 It integrates:

* 🔧 Kinematics (FK & IK)
* 🖥️ Graphical User Interface (GUI)
* 🎯 Simulation & Visualization
* 📡 IoT Control using ESP32
* 👁️ Computer Vision Applications
* 🚀 Trajectory Planning

---

## ✨ Features

* ✅ Forward Kinematics (FK) visualization
* ✅ Inverse Kinematics (IK) solver
* ✅ 3D Robot Simulation
* ✅ Pick & Place Experiment
* ✅ Cup Filling Automation
* ✅ Real-time control via ESP32
* ✅ User-friendly GUI
* ✅ Modular & extendable design

---

## 🛠️ Technologies Used

* 🐍 Python
* 📡 ESP32 Microcontroller
* 📷 ESP32-CAM
* 🎨 Tkinter (GUI)
* 📈 Matplotlib (3D Visualization)
* 🤖 OpenCV (Computer Vision)

## 🚀 How to Run

### 🔹 1. Clone the Repository

```bash
git clone https://github.com/your-username/robot-arm-educational-kit.git
cd robot-arm-educational-kit
```

### 🔹 2. Install Requirements

```bash
pip install -r requirements.txt
```

### 🔹 3. Run the Application

```bash
python main.py
```

## 🧪 Experiments

### 🤖 Forward Kinematics (FK)

Compute end-effector position from joint angles.

### 🎯 Inverse Kinematics (IK)

Find joint angles for target position.

### 📦 Pick & Place

Object manipulation using predefined paths.

### 🥤 Cup Filling System

Automated filling using sensors and control logic.

### 🚀 Trajectory Planning

Generate smooth optimized motion paths between waypoints for stable robot movement.
